# Math-Course-Slides

多门课程的课件, 包括 [实变函数](实变函数), [数学分析](数学分析), 以及 [教学比赛](教学比赛) 相关内容.

## Usage

方法一: 在 [Overleaf](https://www.overleaf.com/) 中打开本项目, 并选择 `XeLatex` compiler.

方法二: 在本地编译, 需要安装 [TeX Live](https://www.tug.org/texlive/) 或者 [MiKTeX](https://miktex.org/), 并使用 `xelatex` 编译器,
或者直接调用 [compile.py](compile.py)

```bash
python compile.py
```
